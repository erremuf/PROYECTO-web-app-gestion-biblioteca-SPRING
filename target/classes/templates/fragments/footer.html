library.app

Esta aplicación es el proyecto final del Máster de Spring y Spring Boot de Ricardo Murillo. Máster impartido por la Ascociación Española de Programadores Informáticos (AEPI). Library App es una aplicación para la gestión de una biblioteca, llevar un control fácilmente de libros, socios y prestamos.
